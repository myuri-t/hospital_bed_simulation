# Hospital Bed Allocation Simulation

This project models hospital inpatient bed allocation using discrete-event simulation to analyze patient flow and hospital resource utilization. The simulation represents the process of patients arriving from the emergency department and being assigned to inpatient wards based on their medical specialty and bed availability.

The goal of the project is to evaluate operational strategies that could reduce patient waiting times and improve overall hospital efficiency. The model captures important operational factors such as patient arrival patterns, treatment durations, discharge approval processes, and bed cleaning times.

Using the simulation model, different scenarios can be tested to understand how changes in hospital policies or resource allocation impact key performance metrics such as patient waiting times, queue lengths, and bed utilization.

## Tools Used
- Simio simulation software
- Python for input modeling and data analysis
- Statistical distribution fitting
- Operations research and simulation techniques
